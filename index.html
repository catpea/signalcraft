Zooming User Interface
efficiency/CPU tests

Fear of the simple act of zooming becoming too CPU intensive on low power computers, when there is a lot on nodes on the screen.

There are however no issues for editing ZUI mode, because only the editor changes size, and only one editor is open at the same time.

  • Click Property You have to click a property to trigger its editor. But it won't work if you are zoomed out, since it would be too hard to see.
  • Scrollwheel You may notice that not all text grows larger when you zoom in. That is not a bug, that is the zooming/zoomable user interface at play. Zooming in under a ZUI means you get more room.

If you can't see text, or need more room to edit it, zoom in with the scroll wheel.
